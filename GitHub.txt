GitHub

Introducao ao GitHub

O que e GitHub?

O GitHub e uma plataforma online que permite hospedar e gerenciar repositories Git. Ele facilita o compartilhamento de codigo, colaboracao entre desenvolvedores e controle de versao

Configuracao do GitHub

git config --global user.name "nome"
git config --global user.email "email@gmail.com"

git remote add origin <ulr> -> conecta ao GitHub na url especifica, ou na sua propria ulr

git clone <url> -> clona um repositorio GitHub no seu repositorio local do Git

git pull -> baixa alteracoes feitas na branch ou naquele repositorio especifico

git fetch -> baixa as mudancas sem aplica-las

git pull -> baixa e aplica imediatamente as mudancas na sua branch atual

git push -u origin master -> envia branch para GitHub

Comandos Avancados

git stash -> salva alteracoes permite guardar alteracoes temporariamente sem fazer um commit

git stash list -> isso exibe uma lista de alteracoes salvas

git stash drop stash@{x} -> onde x e o indice do stash na lista

git stash pop -> restara as alteracoes salvas no stash e remove esse stash da lista

git stats apply stash@{x} -> se quiser aplicar um stash especifico sem remove-lo da lista

git rebase -> permite reorganizar commits ao aplicar mudancas de um branch sobre outro

git cherry-pick <hash> -> esse comando permite copiar um commit de um branch para outro sem trazer todos os commits da sequencia

git revert {hash} -> diferente do git reset, git revert cria um novo commit que inverte as alteracoes de um commit anterior

git reset --hard <hash> -> reverte seu repositorio completamente para um estado anterior, apagando todas as mudancas poasteriores
